# UI GUIDE

## Style Principles
- Use consistent spacing.
- Prefer simple layouts.
- Keep primary actions visually clear.

## Color
- Define primary color usage.
- Define neutral/supporting color usage.
- Reserve destructive color for destructive actions.

## Spacing
- Use a predictable spacing scale.
- Avoid crowded layouts.

## Anti-patterns
- Too many competing primary buttons
- Low-contrast text
- Inconsistent padding
- Hidden state changes without feedback
