# so2x-flow slash commands

These commands are thin wrappers around the installed so2x-flow docs-first workflow.

Available commands:
- `/flow-init`
- `/flow-plan`
- `/flow-feature`
- `/flow-fix`
- `/flow-qa`
- `/flow-review`
- `/flow-evaluate`

Implementation behavior still lives in `.workflow/` and the paired `.claude/skills/*.md` docs.
